Repaso cortito:

  • HTML:

    Lenguaje de marcado, se utilizan etiquetas para decirle al navegador donde debe empezar y donde terminar un elemento. las etiquetas se ponen como <etiqueta> (para apertura) y <etiqueta/> (para cierre)

  • Etiquetas basicas:

    • <!DOCTYPE html> : Marca al navegador el tipo de archivo que usamos

    • <html> </html> : Inicio y fin del codigo html

    • <head> </head> : dentro se engloban metadatos

    • <title> </title> : se enceuntra dentro del head, y representa lo que se lee en la pestaña del navegador

    • <body> </body> : aqui va todo lo visible de nuestra pagina

  • Etiquetas semanticas: sin representacion grafica, utiles para SEO

    • <header> </header> : Para los elementos del encabezado

    • <nav> </nav> : para conjuntos de links que ayuden a navegar el sitio

    • <main> </main> : Lo principal de nuesta pagina

    • <footer> </footer> : para nuestro pie de pagina

    • <section> </section> : todo su contenido está relacionado y forma parte de un mismo significado o elemento

    • <article> </article> : dentro deben haber contenidos únicos con su propio significado

    • <aside> </aside>: Son datos extra sin lo que podríamos pasar perfectamente pero que hemos decidido añadir en el documento.

  • Atributos:

    dentro de la etiqueta de apertura, podemos agregar atributos, en algunos casos son madnatorios apra el funcionamiento de la etiqueta. Los atributos se agregan por su nombre, seguidos de un signo "=" y luego entre comillas el valor del atriburo.
    ej: src="direccion.html"

  • Etiquetas de contenido: Se mostraran como elemntos visibles segun el valor que le demos

    • <h1> </h1> ... <h6> </h6> : Encabezados del 1 al 6, se usan tanto semanticamente como contenido que diferencia improtancia de los titulos

    • <p> </p> : parrafos, texto sin formato

    • <pre> <pre/> : texto preformateado, los cambios que hagamos en los espacios y saltos de linea aplican aqui

    • <br> : break o salto de linea, para bajar un reglon dejando un espacio

    • <hr> : linea horizontal, es un salto de linea que ademas dibuja una linea horizontal como corte de pagina

    • <a> </a> : los elementos dentro de esta etiqueta seran links o hypervinculos hacia la direccion que le indiquemos en el atributo "href"

    • <img> : etiqueta para insertar imagenes, necesita del atributo "src" con un valor de una direccion donde haya una imagen, como buenas practicas se agrega ademas el atributo "alt" con el valor del texto alternativo

    • <figure> : es semantica para indicar un agrupado de una imagen con un caption o una descricion

    • <figcaption> : se utiliza para describir el contenido de una imagen dentro de un figure

    • <ul> </ul> : Unordered List, lista desordenada, dentro de estos tags, cada uno de los elementos llevara un indicativo de donde comienza un item, sin enumeracion.

    • <ol> </ol> : Ordered List, igual que la lista desordenada, pero dentro se enumeran los elementos

    • <li> </li> : el contenido es el de el elemento de una lista

  • Estructura basica:

    la estructura minima que debe llevar una pagina web (decentemente ordenada)

                            <!DOCTYPE html>
                        <html>
                         <head>
                            <title>  </title>
                         </head>
                         <body>
                            <header> </header>
                            <main>   </main>
                            <footer> </footer>
                         </body>
                        </html>

El tag <video> :

Nos permite darle ingresar uno o mas videos dentro en un tag <source>
Y agregar parametros que determinan comportamientos como:

  • width: determina su ancho, ej: width="200px"
  • height: determina la altura, ej: width="150px"
  • controls: Agrega los controles de reproduccion
  • autoplay: hara que (si el navegador lo permite) se autoreproduzca apenas se cargue
  • loop: hara que se reproduzca nuevamente cada vez que termine
  • muted: indica que cuando se cargue , debe empezar silenciado
  • poster: podemos indicar la direccion de una imagen que queramos que aparezca antes de que se reproduzca
  • src: como en un link , nos permite especificar el origen del video para enlazar

Iframe:

Son marcos en los que podemos insertar otras apginas o codigo aparte, y que se represente dentro del mismo.

por ej, un video traido de Youtube
